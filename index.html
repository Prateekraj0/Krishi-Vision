About

Built for farmers,
powered by AI.

PlantCare AI is an open agricultural intelligence platform connecting deep learning with the real needs of crop farmers.

The Problem

Plant diseases cause 20–40% of global crop losses annually, costing farmers hundreds of billions of dollars. Early detection is the single most effective intervention — but most farmers lack access to affordable, fast expert diagnosis when they need it most.

Our Solution

PlantCare AI uses deep learning (TensorFlow/Keras) trained on thousands of annotated leaf images to identify 38+ plant diseases across 12 major crop types. Upload a single photo and receive expert-level diagnosis in under 5 seconds, free of charge.

Model Integration

The core model plant_disease_model.h5 is a convolutional neural network trained on the PlantVillage dataset. Class labels are loaded dynamically from class_names.json, enabling easy expansion to new diseases without code changes. The Flask backend loads the model at startup, preprocesses images to 128×128 px, and returns predictions to the frontend via POST /predict.

Setup Instructions

1. Place plant_disease_model.h5 and class_names.json alongside app.py and index.html.

2. Install dependencies: pip install flask flask-cors tensorflow pillow numpy

3. Run: python app.py

4. Open http://localhost:5000 in your browser.
